## Fast start

    git clone https://github.com/VundleVim/Vundle.vim ~/.vim/bundle/Vundle.vim
    cp vimrc ~/.vimrc
    vim +PluginInstall +qall

## Configure cscope

    cp plugin/*  ~/.vim/plugin/

## Install dependencies

    brew install llvm astyle
